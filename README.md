# Localização Global de Robôs Móveis via AMCL

Este projeto implementa um Filtro de Partículas Adaptativo (AMCL) utilizando a técnica de KLD-Sampling para otimizar a localização de robôs em mapas conhecidos. Desenvolvido com Python e Pygame, o sistema permite simular interativamente o problema do robô sequestrado e a convergência da nuvem de partículas.
O documento pdf incluso neste repositório detalha qual a proposta a ser desenvolvida.
